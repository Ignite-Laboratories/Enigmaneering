Manifold space is an odd thing to consider in a system with infinite lossless compression!

If outcomes could be manifested in parallel, does the universe naturally select for the most _**ideal**_ ones to render?

I'd certainly like to think so!  

From my vantage point - everyone actively listening to the feedback loop of Life is what provisioned this work 
to materialize in the first place.  

Thank you =)

Why?  

Because a shared memory substrate with _motive_ would break _impartiality._

Thus, foundationally, I theorize that such a system would only need to actively discern _the path of least resistance,_
select for it, and actively participate in manifesting that _empathy should always prevail._
